# 🎙️ Speech-to-Text Converter AI

### Convert Audio & Video Files into Text using AI

![Python](https://img.shields.io/badge/Python-3.11-blue?logo=python)
![Streamlit](https://img.shields.io/badge/Streamlit-Web%20App-red?logo=streamlit)
![SpeechRecognition](https://img.shields.io/badge/SpeechRecognition-Audio%20AI-green)
![License](https://img.shields.io/badge/License-MIT-yellow)
![Status](https://img.shields.io/badge/Project-Active-brightgreen)

A modern **AI-powered Speech-to-Text web application** built with **Python & Streamlit** that converts **audio and video files into readable text**.

This project automatically **converts uploaded media files into speech-readable format and extracts transcription**, which can be **downloaded as a `.txt` file**.

# ✨ Features

🚀 Drag & Drop File Upload  
🎧 Supports Audio & Video Formats  
📄 Instant Speech-to-Text Transcription  
⬇️ Download Transcript as `.txt`  
🎨 Modern Dark UI  
⚡ Automatic Audio Conversion

# 📂 Supported File Formats

The system automatically converts files to `.wav` before transcription.

```

ogg
mp4
mov
mkv
```

# 📦 Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Chethumalli/Speech-to-Text-Converter-AI.git
cd AI-Speech-to-Text-Converter
```

---

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 3️⃣ Install FFmpeg

FFmpeg is required for audio conversion.

Download from:

```
https://ffmpeg.org/download.html
```

After installing, add **FFmpeg to your system PATH**.

---

# ▶️ Run the Application

```bash
streamlit run app.py
```

Open in browser:

```
http://localhost:8501
```

---

# ⚙️ How It Works

1️⃣ User uploads an **audio/video file**

2️⃣ The system **converts the file into WAV format**

3️⃣ **SpeechRecognition processes the audio**

4️⃣ Speech is converted into **text transcription**

5️⃣ The extracted text is displayed and can be **downloaded**
